1. 제목
- 점심 추천 도우미

2. 프로젝트 배경 및 목적
- 우리는 대학생으로서 어떤 밥을 먹을까 매일 고민합니다. 그렇기에 점심 추천 도우미를 만들었습니다. 
- 기본적으로 한밭대학교의 학식 및 기숙사식의 데이터를 html로 조사해서 프로그램에 즉시 출력하고 만약 메뉴가 마음에 들지 않는다면, 사용자가 직접 몇 개의 추천을 받을지 설정하고 그에 따라 무작위로 페널에 추천 메뉴를 출력합니다. 

3. 프로젝트에 필요한 이론
- JAVA 기본 문법: 프로그램의 전체적인 로직과 기본 구조를 구현하기 위해 필요합니다.
- JAVA Swing 및 AWT: 사용자가 상호작용할 버튼, 텍스트 영역 등 GUI 화면을 구현하기 위해 필요합니다.
- 웹 크롤링 (HTML 파싱): 한밭대학교 웹사이트에서 학식 및 기숙사식 메뉴 데이터를 가져오기 위해 필요합니다.
- 랜덤 생성 함수: 사용자가 원할 때 무작위로 점심 메뉴를 추천하는 핵심 기능을 구현하기 위해 필요합니다.
- AWT 이벤트 및 리스너: '추천 받기' 등 사용자의 버튼 클릭에 반응하여 특정 동작을 실행하기 위해 필요합니다.
